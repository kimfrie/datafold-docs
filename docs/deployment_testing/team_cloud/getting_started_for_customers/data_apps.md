# Connect Data Apps

:::note
To connect data apps, first connect a [Data Source](/deployment_testing/team_cloud/getting_started_for_customers/data_sources.md), integrate with [Source Control](/deployment_testing/team_cloud/getting_started_for_customers/source_control), and integrate with [dbt](/deployment_testing/team_cloud/getting_started_for_customers/dbt).

Then navigate to **Settings** &rarr; **Data Apps** and click **Add New Integration**.
:::
