# Cloud Reference

### Create an API Key
To generate a Datafold API key, navigate to **Settings** &rarr; **Account** and click **Create API Key**. 

### Set Base URL
- SaaS Customers: `https://app.datafold.com`
- VPC Deployments: `https://datafold.yourdomain.com`
